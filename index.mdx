FLOSS (FLARE Obfuscated String Solver) is an open-source tool from Mandiant's FLARE team that goes beyond traditional string extraction. While `strings.exe` only finds plaintext strings, FLOSS uses advanced static analysis and CPU emulation to automatically recover strings that malware deliberately hides — including stack strings, tight strings, and cryptographically decoded strings.

## What FLOSS extracts

FLOSS identifies and recovers four distinct categories of strings that malware uses to hide indicators of compromise:

<CardGroup cols={2}>
  <Card title="Static strings" icon="file-lines">
    Classic ASCII and UTF-16LE strings embedded in the binary — the same strings `strings.exe` finds. FLOSS provides a baseline that everything else is compared against.
  </Card>
  <Card title="Stack strings" icon="layer-group">
    Strings assembled character-by-character on the stack at runtime. FLOSS emulates function execution to reconstruct them without running the malware.
  </Card>
  <Card title="Tight strings" icon="arrows-spin">
    A specialized form of stack strings decoded through tight loops. FLOSS handles these separately using loop-aware emulation to avoid false positives.
  </Card>
  <Card title="Decoded strings" icon="key">
    Strings encrypted in global memory or on the heap that are decrypted by dedicated decoding functions. FLOSS identifies decoders via heuristics and emulates them.
  </Card>
</CardGroup>

## Language-aware extraction

FLOSS automatically detects Go and Rust binaries, where strings are stored in non-NULL-terminated formats that confuse standard tools. For these languages, FLOSS applies language-specific algorithms to reconstruct strings as they appear in the original source code.
